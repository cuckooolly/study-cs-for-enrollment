# 컴퓨터 과학은 키워드 암기 과목이 아니다

- 실제 기술면접에서는 원리를 물어보기 때문
  - 예시) 웹 브라우저에 URL을 입력하고, 웹 페이지가 구현되기까지의 과정을 설명해보세요.

- 실무에서 문제를 해결하기 위해서는, 프로그램의 실행 원리를 이해해야 한다.
  - 예시) 잘 되던 네트워크가 끊어졌을 때는, 어떻ㅎ게 문제를 해결할 수 있을 것인가?
