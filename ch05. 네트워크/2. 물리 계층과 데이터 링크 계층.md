# 물리 계층과 데이터 링크 계층

## 이더넷

- 다양한 통신 매체의 규격들과 송수신 되는 프레임의 형태, 프레임을 주고 받는 방법 등이 정의된 네트워크 기술

### 이더넷 표준

- IEEE 802.3: 이더넷 표준에 따라 지원되는 네트워크 장비, 통신 매체의 종류와 전송 속도 등이 달라질 수 있음
  - 10BASE-T: 최대 10Mbps, 트위스티드 페어 케이블 사용
  - 100BASE-TX: 최대 100Mbps, 트위스티드 페어 케이블 사용
  - 1000BASE-T: 최대 1Gbps, 트위스티드 페어 케이블 사용
  - 10GBASE-T: 최대 10Gbps, 트위스티드 페어 케이블 사용
  - 10GBASE-SR: 최대 10Gbps, 광섬유 케이블 사용

### 이더넷 프레임

- 이더넷 네트워크에서 주고받는 프레임
- 구성 요소
  - 프리앰블: 수신 장치가 프레임의 시작을 인식할 수 있도록 도와주는 역할
  - 목적지 MAC 주소: 프레임을 수신할 장치의 물리적 주소
  - 출발지 MAC 주소: 프레임을 전송한 장치의 물리적 주소
    - MAC 주소: 네트워크 인터페이스마다 부여되는 6바이트(48비트) 길이의 주소로 LAN 내의 수신지와 송신지를 특정할 수 있음
  - 유형: 상위 계층 프로토콜을 식별하는 필드 (예: IPv4, IPv6)
  - 데이터: 실제 전송되는 데이터, 최대 1500바이트 (MTU)
  - FCS (Frame Check Sequence): 오류 검출을 위한 필드

## 유무선 통신 매체

### 유선 매체 - 트위스티드 페어 케이블

- 트위스티드 페어 케이블: 구리 선으로 전기 신호를 주고받는 통신 매체로 케이블 본체와 커넥터로 구성됨
- RJ-45: 트위스티드 페어 케이블에서 활용되는 커넥터
- 노이즈: 외부 전자기파 간섭으로 인해 신호가 왜곡되는 현상
- 차폐: 외부 전자기파 간섭을 줄이기 위해 케이블을 감싸는 것
- 종류
  - STP (Shielded Twisted Pair): 차폐가 된 트위스티드 페어 케이블
  - UTP (Unshielded Twisted Pair): 차폐가 되지 않은 트위스티드 페어 케이블
  - FTP (Foiled Twisted Pair): 알루미늄 호일로 차폐된 트위스티드 페어 케이블

### 무선 매체 - 전파와 Wi-Fi

- Wi-Fi: 무선 주파수를 이용해 데이터를 주고받는 무선 통신 기술
- 주파수 대역: Wi-Fi에서 주로 사용하는 2.4GHz와 5GHz 대역
- Wi-Fi 표준: IEEE 802.11 시리즈로 구성됨
- 채널: 주파수 대역 내에서 데이터를 주고받기 위해 할당된 구간
- AP: 무선 액세스 포인트, 무선 장치가 유선 네트워크에 접속할 수 있도록 해주는 장비

## 네트워크 인터페이스: NIC

- NIC: 호스트와 통신 매체를 연결하고, MAC 주소가 부여되는 네트워크 장비
- 요즘에는 마더보드에 내장된 경우가 많음

## 허브와 스위치

### 물리 계층의 허브

- 허브: 여러 대의 호스트를 연결하는 물리 계층의 네트워크 장비
  - 전달받은 신호를 모든 포트로 내보냄
  - 반이중 모드(한 방향으로만 통신)로만 통신이 가능
- 리피터: 네트워크의 전기 신호를 증폭시켜 전송 거리를 늘려주는 장비

### 데이터 링크 계층의 스위치

- 스위치: 여러 대의 호스트를 연결하는 데이터 링크 계층의 네트워크 장비
  - MAC 주소 테이블을 활용해 목적지 MAC 주소에 해당하는 포트로만 프레임을 전달
  - 전이중 모드(양방향으로 통신)로 통신이 가능
- VLAN: 논리적으로 분리된 네트워크 세그먼트를 구성하는 기술
