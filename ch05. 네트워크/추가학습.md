# 소켓 프로그래밍

- 소켓
  - 프로세스 간 네트워크 통신의 엔드포인트 
  - 많은 운영체제에서 소켓은 파일과 같은 방식으로 다룸
  
    <br>
    
    ![](https://csnote.net/assets/img/net/socket.png)
  
- 서버 프로그램은 소켓 관련 네트워크 시스템 콜을 호출하여, 네트워크 기능을 사용

![](https://img1.daumcdn.net/thumb/R1280x0/?scode=mtistory2&fname=https%3A%2F%2Fblog.kakaocdn.net%2Fdna%2FcI7lx8%2FbtrzQZ1OJDi%2FAAAAAAAAAAAAAAAAAAAAAHGqKh8Ao6CslKB9TqDEkj9UR08RA9F1xcSR1Pn-G08C%2Fimg.jpg%3Fcredential%3DyqXZFxpELC7KVnFOS48ylbz2pIh7yKj8%26expires%3D1767193199%26allow_ip%3D%26allow_referer%3D%26signature%3DWgzScXcc8A%252Fq6e0GSxFEiXNd8pg%253D)

- 간단한 Python 소켓 프로그래밍 예제

```python
# echo-server.py
import socket

HOST = "127.0.0.1"  # Standard loopback interface address (localhost)
PORT = 65432  # Port to listen on (non-privileged ports are > 1023)

with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
    s.bind((HOST, PORT)) # 소켓을 특정 네트워크 인터페이스 및 포트번호와 연결하는 데 사용
    s.listen() # 소켓이 들어오는 연결을 수신 대기하도록 설정
    conn, addr = s.accept() # 들어오는 연결을 수락하고 새로운 소켓 객체와 클라이언트 주소를 반환
    with conn: # 연결된 소켓 객체를 사용하여 클라이언트와 통신
        print(f"Connected by {addr}") 
        while True:
            data = conn.recv(1024)
            if not data:
                break
            conn.sendall(data) # 받은 데이터를 다시 클라이언트로 전송
```

```python
# echo-client.py
import socket

HOST = "127.0.0.1"  # The server's hostname or IP address
PORT = 65432  # The port used by the server

with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s: # 소켓 객체 생성
    s.connect((HOST, PORT)) # 서버에 연결
    s.sendall(b"Hello, world") # 서버로 데이터 전송
    data = s.recv(1024) # 서버로부터 데이터 수신

print(f"Received {data!r}")
```
